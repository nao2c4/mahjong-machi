# mahjong-machi
門前清一色待ち一覧

## 表の見方

例: 
```
1112345678999p: 123456789p (23)
```

`1112345678999p`という手配だった場合，`123456789p`が待ちになり残っている枚数は`23`枚という意味．
なお，`111999p11z`は一筒が3枚，九筒が3枚，東が2枚という意味．
